### 분류
- 조건문

### 문제 요약
- ineq는 "<"와 ">"중 하나, eq는 "="와 "!"중 하나, 두 정수 n과 m 매개변수
- ">", "=" : n >= m
- "<", "=" : n <= m
- ">", "!" : n > m
- "<", "!" : n < m
- 위의 조건을 만족하면 1 아니먄 0 리턴

### 풀이 아이디어
- if 조건문 이용
- 비교 연산자 == 이용
